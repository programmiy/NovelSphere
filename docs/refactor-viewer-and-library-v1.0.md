# 리팩토링: 뷰어 기능 및 서재 UI 정리 (v1.0)

## 변경 사항

- 뷰어 페이지의 사용성을 개선하고, 서재 페이지의 중복 기능을 정리했습니다.

### 1. 뷰어 기능 추가 및 개선

- **집중 모드:** 헤더 UI를 최소화하여 본문에 집중할 수 있는 기능을 추가하고, 해당 모드를 쉽게 켜고 끌 수 있도록 버튼 동작을 개선했습니다.
- **원문 보기:** 뷰어 내에서 원문을 함께 볼 수 있는 토글 기능을 추가했습니다.
- **드래그 즉시 수정:** 뷰어에서 텍스트를 선택하여 바로 수정하고 저장할 수 있는 기능을 구현했습니다.
- **스타일 및 캐시:** 뷰어 페이지의 디자인을 개선하고, CSS 캐시 문제를 해결했습니다.

### 2. 서재 UI 정리

- '원문 보기' 기능이 뷰어로 이전됨에 따라, 서재 페이지에 있던 중복된 '원문 함께 보기 (뷰어)' 토글 스위치와 관련 로직을 모두 제거하여 UI를 간소화했습니다.
