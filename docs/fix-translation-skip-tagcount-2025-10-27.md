# [Fix] pid가 'tagCount'일 경우 번역 제외

## 변경 사항

- `server.py`의 `translate_texts` 함수를 수정하여, 번역 요청 시 `pid` (API 요청 시 `id` 필드에 해당)가 'tagCount'인 항목을 번역에서 제외하도록 변경했습니다.
- 해당 항목은 번역 API로 전송되지 않고, 원문 텍스트가 그대로 번역 결과로 반환됩니다.

## 목적

- 특정 조건('tagCount' pid)을 가진 항목이 불필요하게 번역되는 것을 방지하여 API 사용량을 절감하고, 의도치 않은 번역 오류를 예방합니다.
