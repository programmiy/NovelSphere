# UI 가시성 수정 (v1.0)

## 개요
사용자 피드백에 따라 번역 기록 페이지의 UI 가시성 문제를 해결했습니다. 상단 컨트롤 버튼이 보이지 않던 문제와 번역 기록이 표시되지 않던 문제를 수정했습니다.

## 주요 변경 사항

### 1. 상단 컨트롤 버튼 가시성 수정
- `controls-bar` 내 `bulk-actions` div의 위치가 레이아웃 문제를 일으켜 상단 버튼의 일부가 보이지 않던 문제를 해결했습니다.
- `history.html`에서 `bulk-actions` div를 `controls-bar` 외부로 이동하여 독립적인 행을 차지하도록 변경했습니다. 이로써 `controls-bar`의 flex 레이아웃이 방해받지 않아 모든 상단 버튼이 올바르게 표시됩니다.

### 2. 번역 기록 표시 문제 수정
- `history.js`가 번역 기록을 동적으로 렌더링할 때 `history.html`에 미리 정의된 구조와 충돌하여 기록이 표시되지 않던 문제를 해결했습니다.
- `history.html`의 `history-container` 내부에서 미리 존재하던 `list-group-content` 및 `history-table` 구조를 제거했습니다. 이제 `history-container`는 `history.js`가 동적으로 콘텐츠를 채울 빈 플레이스홀더 역할을 합니다.

## 결과
- 모든 상단 컨트롤 버튼이 페이지에 올바르게 표시됩니다.
- 로컬 스토리지에 저장된 번역 기록이 페이지에 정상적으로 로드되고 표시됩니다.

이로써 사용자가 보고 있던 주요 UI 가시성 문제가 해결되었습니다.
