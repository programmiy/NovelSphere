# 상단 컨트롤 바 통합 리팩토링 v1.0

## 변경 사항

- `history.html` 상단의 여러 제어 섹션(검색, 폴더 생성, 통계 등)을 하나의 통합된 수평 컨트롤 바로 통합했습니다.
- `.master-controls`라는 새로운 Flexbox 컨테이너를 도입하여 모든 제어 관련 컴포넌트를 자식으로 포함하도록 HTML 구조를 리팩토링했습니다.
- 기존의 개별 섹션 스타일을 제거하고, 새로운 부모 컨테이너와 자식 wrapper들에 스타일을 재정의하여 일관된 디자인을 적용했습니다.

## 영향

- 번역 기록 페이지 상단의 UI가 여러 개의 분리된 섹션에서 하나의 깔끔한 컨트롤 바로 변경되어 레이아웃이 개선되고 사용성이 향상됩니다.
