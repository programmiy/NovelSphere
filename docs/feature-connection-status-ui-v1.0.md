# 기능 추가: 연결 상태 UI 표시 (v1.0)

## 변경 사항

- **CSS 수정 (`admin.css`, `library.css`, `viewer.css`):**
  - 네트워크 연결 상태(온라인/오프라인)를 표시하기 위한 공통 토스트(Toast) 알림 스타일을 추가했습니다.

- **JS 수정 (`admin.js`, `library.js`, `viewer.js`):**
  - 브라우저의 `online` 및 `offline` 이벤트를 감지하는 이벤트 리스너를 추가했습니다.
  - 네트워크 상태가 변경되면, 화면 하단에 "온라인 상태입니다." 또는 "오프라인 상태입니다." 메시지를 표시하는 토스트 알림 함수를 구현했습니다.

## 기대 효과

- 사용자가 현재 네트워크 연결 상태를 직관적으로 인지할 수 있습니다.
- 오프라인 상태일 때 PWA의 캐싱 기능이 동작하고 있음을 사용자에게 알려주어, 기능이 제한될 수 있음을 안내합니다.
