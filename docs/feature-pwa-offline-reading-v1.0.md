# 기능: PWA 오프라인 읽기 기능 구현 (v1.0)

## 변경 사항

- 로드맵의 마지막 항목인 PWA(Progressive Web App) 오프라인 읽기 기능을 구현했습니다.
- 이제 사용자가 한 번 이상 열어본 책은 인터넷 연결 없이도 뷰어에서 다시 읽을 수 있습니다.

### Service Worker 수정 (`service-worker.js`)

- **캐싱 전략 변경:** 기존에는 모든 API 요청을 네트워크로만 전달했지만, 이제 `/stream` 엔드포인트(책 데이터)에 대한 요청을 캐시하도록 수정했습니다.
- **네트워크 우선, 캐시 폴백 (Network-First, Cache-Fallback):**
  1. 사용자가 책을 열면, 먼저 네트워크를 통해 최신 데이터를 가져오려고 시도합니다.
  2. 데이터를 성공적으로 가져오면, 응답을 캐시에 저장하고 사용자에게 보여줍니다.
  3. 네트워크 연결에 실패하면(오프라인 상태), 캐시에 저장된 데이터를 대신 보여줍니다.
- 이를 통해 온라인 상태에서는 항상 최신 데이터를 보고, 오프라인 상태에서는 이전에 캐시된 데이터를 볼 수 있습니다.
