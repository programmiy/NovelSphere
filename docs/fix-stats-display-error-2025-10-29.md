
# 기능: 관리자 페이지 통계 표시 오류 수정

## 개요

관리자 페이지의 통계 섹션(총 번역, 오늘 번역, 평균 길이)이 항상 0으로 표시되는 오류를 수정했습니다.

## 원인

서버에서 전달하는 데이터의 구조가 최근 변경되었으나, 통계를 계산하는 클라이언트 사이드 JavaScript(`admin.js`)의 `updateStatistics` 함수가 이전 데이터 구조를 그대로 사용하고 있어 발생한 문제였습니다.

- **기존 데이터 구조**: `groupedData[folderName][url]`
- **변경된 데이터 구조**: `groupedData[folderName]`는 `urlGroup` 객체의 배열이며, 각 `urlGroup`은 `items` 배열을 포함합니다.

## 변경 사항

- **`server/static/admin.js` 수정**:
  - `updateStatistics` 함수 내부의 반복문 로직을 변경된 데이터 구조에 맞게 수정했습니다.
  - 이제 함수는 `urlGroup` 배열을 올바르게 순회하고, 각 그룹 내의 `items` 배열에 접근하여 통계를 정확하게 계산합니다.

## 기대 효과

- 관리자 페이지에 총 번역 수, 오늘 번역 수, 평균 번역 길이가 정상적으로 표시됩니다.
