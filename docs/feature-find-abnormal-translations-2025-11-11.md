# 기능: '이상번역 찾기' 기능 개선

## 개요

Admin 페이지의 '빈번역 찾기' 기능의 라벨을 '이상번역 찾기'로 변경하고, 기능을 확장하여 번역문에 일본어가 포함된 경우도 함께 찾아내도록 개선했습니다. 모든 로직은 클라이언트 사이드(`admin.js`)에서 처리됩니다.

## 주요 변경 사항

- **UI 변경**:
  - `admin.html` 템플릿의 '빈번역 찾기' 버튼 라벨을 '이상번역 찾기'로 변경했습니다.

- **필터링 로직 확장**:
  - `admin.js`의 필터링 로직을 수정하여 다음 두 가지 조건 중 하나에 해당하는 번역을 '이상번역'으로 간주하도록 변경했습니다.
    1.  번역문이 비어있거나 공백으로만 이루어진 경우 (기존 기능)
    2.  번역문에 일본어 문자가 포함된 경우 (신규 기능)
  - 일본어 감지를 위해 정규식을 사용합니다.

- **사용자 알림 메시지 변경**:
  - 필터링 결과 '이상번역'이 없을 경우, "이상 번역 기록이 없습니다."라는 알림 메시지가 표시되도록 수정했습니다.
