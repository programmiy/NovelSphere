# 기능: 사이드바 레이아웃 개선 (v1.0)

## 변경 사항

- 번역 사이드바가 페이지에 가로 스크롤바를 유발하는 문제를 해결하기 위해 레이아웃을 개선했습니다.

### CSS 수정 (`content.js`)

- **유연한 너비:** 사이드바의 고정 너비(`width: 450px`)를 유지하되, 화면 크기에 따라 조절될 수 있도록 `max-width: 90vw`와 `min-width: 300px`를 추가했습니다.
- **애니메이션 개선:** 사이드바가 나타나고 사라지는 애니메이션을 `right` 속성 대신 `transform: translateX()`를 사용하도록 변경하여, 너비가 유동적이어도 부드럽게 동작하도록 개선했습니다.
- **테이블 레이아웃 고정:** 사이드바 내부의 번역 테이블에 `table-layout: fixed`를 적용하고, 각 열(원문, 번역, 저장)에 백분율(%) 너비를 지정했습니다. 이를 통해 내용이 길어져도 테이블이 사이드바를 넘어가지 않고, 셀 내부에서 텍스트가 올바르게 줄바꿈되도록 수정했습니다.
