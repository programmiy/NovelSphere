# 폴더 미표시 버그 수정 v1.2

## 개요

관리자 페이지에서 새로 생성한 폴더가 즉시 표시되지 않는 지속적인 버그를 해결했습니다.

## 원인

브라우저가 폴더 목록을 가져오는 API의 응답을 캐시(저장)하여, 폴더 생성 직후에도 새로운 목록을 가져오지 않고 이전에 저장된 데이터를 사용하여 문제가 발생했습니다.

## 수정 사항

- **`server.py`:**
  - `/folders` 및 `/translations_by_folder` API 엔드포인트의 응답에 `Cache-Control: no-cache` 헤더를 추가했습니다.
  - 이 헤더는 브라우저가 해당 API의 응답을 캐시하지 않고 매번 새로 요청하도록 강제하여, 항상 최신 폴더 목록을 받아오게 합니다.
