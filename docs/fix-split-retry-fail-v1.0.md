# Fix: 분할 재시도 실패 처리 수정 (v1.0)

## 개요

번역 과정에서 문장 분할 재시도까지 최종적으로 실패했을 때의 사용자 경험을 개선합니다.

## 변경 사항

-   `PROHIBITED_CONTENT` 등의 이유로 문장 분할 재시도 후에도 번역에 실패하는 경우, 기존의 `번역 실패: 분할 재시도 실패` 오류 메시지를 번역 결과에 표시하는 대신 빈 텍스트를 삽입합니다.
-   번역 실패가 발생한 경우, 해당 문서의 제목(`document.title`) 맨 앞에 `[번역 실패: 분할 재시도 실패 + 실패한 pid 위치:(해당 pid)]` 형식의 태그를 추가합니다.
-   이 변경된 제목은 번역 기록 저장 시 함께 저장되어, 사용자가 나중에 어떤 문서에서 오류가 발생했는지 쉽게 식별할 수 있도록 합니다.

## 영향

-   **`extension/content.js`**: `retryWithSentenceSplitting` 함수의 `catch` 블록 로직을 수정하여, 실패 시 빈 텍스트를 표시하고 `document.title`을 변경하도록 수정했습니다.
