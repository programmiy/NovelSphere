# Fix: 관리자 페이지 UI 반응성 개선 (v1.0)

## 설명

관리자 페이지(`admin.html`)의 UI가 모바일 기기에서 올바르게 표시되지 않는 문제를 해결했습니다.

## 변경 사항

- **`server/static/admin.css`**:
  - `@media` 쿼리를 추가하고 수정하여 다양한 화면 크기(1200px, 768px, 480px)에 대응하도록 반응형 디자인을 개선했습니다.
  - 작은 화면에서는 컨트롤 바, 폴더 관리, 통계 영역이 세로로 쌓이도록 변경했습니다.
  - 모바일 화면에서의 가독성을 높이기 위해 폰트 크기와 패딩을 조정했습니다.
