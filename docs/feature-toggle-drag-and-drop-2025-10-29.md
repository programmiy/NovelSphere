
# 기능: 드래그 앤 드롭 기능 토글

## 개요

사용자가 관리자 페이지에서 URL 그룹의 순서를 드래그 앤 드롭으로 변경하는 기능을 켜고 끌 수 있도록 토글 스위치를 추가했습니다.

## 변경 사항

- **UI 추가**:
  - `server/templates/admin.html` 파일의 제어판에 "드래그앤 드롭 순서 변경" 체크박스를 추가했습니다.
  - 이 체크박스는 기본적으로 비활성화(체크 해제)되어 있습니다.

- **JavaScript 로직 수정**:
  - `server/static/admin.js` 파일에 다음 로직을 추가했습니다.
    - `Sortable.js` 인스턴스를 저장하고 관리하기 위한 `sortableInstances` 배열을 추가했습니다.
    - 페이지가 렌더링될 때, 체크박스의 상태에 따라 `Sortable.js` 기능의 활성화/비활성화 상태를 초기화합니다.
    - 체크박스의 상태가 변경될 때마다, 모든 `Sortable.js` 인스턴스의 `disabled` 옵션을 동적으로 변경하여 드래그 앤 드롭 기능을 제어합니다.
    - 기능의 상태가 변경될 때 사용자에게 토스트 메시지로 상태를 알려줍니다.

## 기대 효과

- 사용자는 필요에 따라 드래그 앤 드롭 기능을 활성화하여 사용할 수 있습니다.
- 기본적으로 비활성화되어 있어, 실수로 순서가 변경되는 것을 방지하고 원치 않는 UI 상호작용을 줄여 사용자 경험을 개선합니다.
