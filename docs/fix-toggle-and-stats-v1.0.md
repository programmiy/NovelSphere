# 기본 토글 상태 및 통계 오류 수정 v1.0

## 변경 사항

- **기본 토글 상태 수정:**
  - 번역 기록 페이지 로드 시 모든 폴더 및 URL 그룹이 기본적으로 접힌 상태로 표시되도록 `renderGroupedView` 함수의 로직을 수정했습니다.
  - `.collapsed` 클래스를 기본으로 추가하고 토글 아이콘을 '▶'로 변경하여 초기 UI 상태의 일관성을 확보했습니다.

- **통계 오류 수정:**
  - 통계 수치가 모두 0으로 표시되던 문제를 해결하기 위해, 리팩토링 과정에서 누락되었던 `updateStats` 함수를 `history.js`에 다시 추가했습니다.

## 영향

- 이제 페이지 로드 시 모든 기록 그룹이 깔끔하게 접힌 상태로 표시됩니다.
- 총 번역, 오늘 번역, 평균 길이 통계가 정상적으로 계산되어 표시됩니다.
