# 기능: 데이터베이스 자동 백업 (v1.0)

## 개요

서버의 누적 실행 시간을 기준으로 `translations.db` 파일을 주기적으로 자동 백업하는 기능을 추가했습니다.

## 주요 로직

1.  **누적 실행 시간 기록**:
    *   서버의 총 누적 실행 시간을 `server/server_meta.json` 파일에 초 단위로 저장합니다.
    *   서버 시작 시 이 값을 읽어오고, 서버 종료 시 현재 세션의 실행 시간을 더하여 다시 저장합니다.

2.  **백그라운드 작업**:
    *   서버가 시작되면 10분마다 누적 실행 시간을 확인하는 백그라운드 작업이 실행됩니다.

3.  **백업 트리거**:
    *   누적 실행 시간이 24시간을 초과하면 백업이 실행됩니다.
    *   `server/backups/` 디렉토리에 `translations_backup_{타임스탬프}.db` 형식으로 데이터베이스 파일의 복사본을 생성합니다.
    *   백업이 성공적으로 완료되면, 누적 실행 시간 카운터는 0으로 초기화됩니다.
