# Fix: admin.js 구문 오류 수정 (v1.0)

## 설명

`admin.js` 파일의 `handleHistoryClick` 함수 내에서 제목을 수정하는 로직에 구문 오류가 있었습니다. 템플릿 리터럴의 닫는 괄호가 누락되어 특정 상황에서 자바스크립트 오류가 발생했습니다.

## 변경 사항

- `handleHistoryClick` 함수 내에서 `titleElement.textContent`를 업데이트하는 두 곳에 누락된 닫는 괄호 `)`를 추가하여 구문 오류를 수정했습니다.

## 영향

- 관리자 페이지에서 제목 수정 기능이 오류 없이 정상적으로 동작합니다.
