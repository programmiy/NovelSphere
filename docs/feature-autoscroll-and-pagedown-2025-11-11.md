# 기능: 번역 완료 후 지연 자동 스크롤 및 PageDown

## 개요

`content.js`의 자동 스크롤 기능이 사용자의 최종 요구사항에 맞춰 개선되었습니다. 번역이 성공적으로 완료되면, 2초 대기 후 페이지 끝까지 자동으로 스크롤하고, 'PageDown' 키를 한 번 누른 뒤 모든 관련 동작을 종료합니다.

## 주요 변경 사항

- **지연 실행**:
  - 번역 완료 후, `setTimeout`을 사용하여 2초의 대기 시간을 가집니다.
  - 이를 통해 사용자가 번역 완료 상태를 인지하고 다음 동작을 예상할 수 있습니다.
- **단일 전체 스크롤**:
  - 2초 대기 후, `window.scrollTo`를 이용해 페이지 맨 아래로 부드럽게 한 번에 스크롤합니다.
  - 기존의 주기적인 스크롤 방식(`setInterval`)은 완전히 제거되었습니다.
- **스크롤 완료 감지**:
  - 부드러운 스크롤이 완료되는 시점을 감지하기 위해, 짧은 주기로 스크롤 위치를 확인하는 로직이 사용됩니다.
- **단일 'PageDown' 실행 및 종료**:
  - 스크롤이 페이지 끝에 도달하면, 'PageDown' 키보드 이벤트를 한 번 발생시켜 웹사이트의 다음 페이지 로드 기능을 트리거합니다.
  - 'PageDown' 이벤트 발생 후 모든 자동 스크롤 관련 동작은 완전히 종료됩니다.
