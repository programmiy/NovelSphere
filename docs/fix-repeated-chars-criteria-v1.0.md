# 수정: 반복 문자 감지 기준 조정 (v1.0)

## 변경 사항

- `server/server.py`의 `process_repeated_chars` 함수 수정
- 번역 후처리 과정에서 반복 문자를 `(반복)`으로 치환하는 기준을 기존 '4회 이상'에서 '6회 이상'으로 변경했습니다.
- 정규표현식을 `(.)\1{3,}`에서 `(.)\1{5,}`으로 수정했습니다.

## 변경 목적

- 기존 기준이 너무 엄격하여, 의성어나 감탄사 등 의도된 표현까지 `(반복)`으로 치환되는 문제를 완화하기 위함입니다.
- 이로 인해 불필요하게 데이터가 오염되는 것을 줄이고 번역의 가독성을 높입니다.
