# 기능: 기호/숫자 반복 자동 복원 스크립트 추가 (v1.0)

## 개요

번역 과정에서 생성된 `(반복)` 태그 중, 원문이 기호 또는 숫자 반복인 경우에 한해 원본 문자열로 자동 복원하는 `server/map_and_fix_repeats.py` 스크립트를 추가했습니다.

## 주요 기능 및 로직

1.  **1:1 매핑 기반 복원**
    *   원본 텍스트에서 4회 이상 반복되는 모든 구간(글자, 기호, 숫자)의 순서와, 번역문의 `(반복)` 태그 순서를 1:1로 매핑합니다.
    *   원본의 N번째 반복 구간이 기호/숫자일 경우에만, 번역문의 N번째 `(반복)` 태그를 원본 문자열(예: `!!!!`)로 교체합니다.
    *   원본이 글자 반복일 경우, `(반복)` 태그를 그대로 유지하여 사용자가 직접 수정할 수 있도록 합니다.

2.  **`六点リーダー` (`……`) 특별 처리**
    *   일반적인 4회 반복 규칙과 별개로, `……` 문자열을 하나의 독립된 '기호' 패턴으로 감지하는 특별 규칙을 추가했습니다.

3.  **개수 불일치 시 예외 처리**
    *   원본의 반복 패턴 수와 번역문의 `(반복)` 태그 수가 일치하지 않을 경우, 기본적으로는 수정을 건너뜁니다.
    *   단, 이 경우에도 원본에 `……`가 포함되어 있다면, 잘못된 번역일 가능성을 고려하여 해당 번역문의 모든 `(반복)` 태그를 `……`로 일괄 교체하는 예외 로직을 적용합니다.

4.  **상세 로그 출력**
    *   스크립트 실행 시, 수정되는 모든 항목에 대해 폴더명, 제목, PID, 원본, 수정 전/후 내용을 상세히 출력하여 사용자가 변경 내역을 쉽게 확인할 수 있도록 했습니다.
