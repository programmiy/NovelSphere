# 아코디언 기본 상태 되돌리기 (v1.0)

## 개요
사용자 피드백에 따라 번역 기록 페이지의 아코디언 그룹 기본 상태를 다시 '접힌 상태'로 되돌렸습니다. 이전에는 기본적으로 열린 상태로 변경했으나, 사용자의 의도는 접힌 상태로 시작하는 것이었습니다.

## 주요 변경 사항

### 1. 아코디언 그룹 기본 상태 변경 (접힌 상태)
- `history.js` 파일의 `renderView` 함수에서 아코디언 그룹을 렌더링할 때, `list-group-toggle` 아이콘을 `▲`에서 `▼`로 다시 변경했습니다.
- 또한, `list-group-content` 요소에 `collapsed` 클래스를 다시 추가하여 페이지 로드 시 번역 기록 그룹이 기본적으로 접힌 상태로 표시되도록 했습니다.

## 결과
- 번역 기록 그룹이 페이지 로드 시 기본적으로 접힌 상태로 표시됩니다.

이 수정으로 사용자 의도에 맞는 아코디언 기본 동작이 복구되었습니다.
