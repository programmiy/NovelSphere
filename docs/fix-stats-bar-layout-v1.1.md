# 통계 바 레이아웃 수정 v1.1

## 변경 사항

- `history.html`의 통계 바가 상단 컨트롤 바와 시각적으로 구분되지 않고 내부 아이템 정렬이 불안정한 문제를 수정했습니다.
- `.stats-bar`의 배경색을 `--card-bg`에서 `--group-header-bg`로 변경하여 컨트롤 바와 시각적으로 구분되도록 했습니다.
- `.stat-item`의 스타일을 `text-align: center`에서 `display: flex` 기반의 column 정렬로 변경하여 숫자와 라벨이 항상 명확하게 중앙 정렬되도록 수정했습니다.

## 영향

- 이제 통계 바는 다른 UI 요소와 명확히 구분되며, 통계 수치들이 항상 중앙에 올바르게 표시됩니다.
