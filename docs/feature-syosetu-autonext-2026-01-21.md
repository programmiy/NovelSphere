# 기능: Syosetu 사이트 자동 다음 페이지 기능

## 변경 사항

- `extension/content.js` 파일을 수정하여 `novel18.syosetu.com` 사이트에서 번역이 완료된 후 다음 페이지로 자동으로 넘어가는 기능을 개선했습니다.
- 기존의 'PageDown' 키 입력 방식 대신, 페이지 내의 '次へ' (다음) 버튼을 찾아 자동으로 클릭하도록 변경했습니다.
- 만약 '次へ' 버튼을 찾지 못할 경우, 기존과 같이 'PageDown' 이벤트를 발생시켜 하위 호환성을 유지합니다.
- 이 기능은 `novel18.syosetu.com` 도메인에서만 활성화되며, 다른 사이트에서는 기존의 자동 스크롤 및 'PageDown' 방식이 유지됩니다.

## 기대 효과

- 사용자가 `novel18.syosetu.com`에서 번역 기능을 사용할 때, 수동으로 다음 페이지로 이동할 필요 없이 자동으로 다음 화가 로드되어 사용자 경험이 향상됩니다.
