# 기능: 뷰어 UX 개선

## 버전

v1.0

## 변경 사항

- **호버 UI 개선:**
  - 뷰어의 좌우 탐색 영역에 마우스를 올리면, 반투명한 그래디언트 배경과 함께 화살표가 나타나도록 하여 클릭 가능한 영역임을 더 명확하게 암시합니다.

- **페이지 넘김 효과 추가:**
  - 페이지를 넘길 때, 콘텐츠가 부드럽게 사라지고 나타나는 크로스페이드 애니메이션을 추가하여 시각적인 경험을 향상했습니다.
