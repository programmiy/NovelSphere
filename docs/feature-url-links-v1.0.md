# 번역 기록 URL 링크 기능 추가 v1.0

## 변경 사항

- 번역 기록 페이지에서 각 URL 그룹의 제목을 클릭하면 해당 원본 페이지로 이동하는 기능이 누락된 문제를 해결했습니다.
- `history.js`의 `renderGroupedView` 함수를 수정하여 그룹 제목을 원본 URL로 연결되는 `<a>` 태그로 감쌌습니다.
- `history.html`에 링크에 대한 기본 스타일을 추가하여 UI 일관성을 유지했습니다.

## 영향

- 이제 사용자는 번역 기록 목록에서 각 그룹의 제목을 클릭하여 손쉽게 원본 컨텍스트를 확인할 수 있습니다.
