# 기능 추가: 번역 기록에 PID 표시 (v1.0)

## 변경 사항 요약
번역 기록 페이지의 각 번역 항목에 고유 식별자(PID)를 표시하는 새로운 열을 추가했습니다. 이를 통해 사용자는 각 번역의 원본 위치를 더 쉽게 식별하고 추적할 수 있습니다.

## 상세 내용

### 프론트엔드 (`extension/history.js`)
1.  **테이블 헤더 추가**:
    *   `renderGroupedHistory` 함수 내에서 번역 기록을 표시하는 테이블의 헤더(thead)에 'PID'라는 새로운 열(`<th>`)을 추가했습니다.

2.  **테이블 셀 추가**:
    *   `renderItem` 함수에서 각 번역 항목(row)을 생성할 때, 체크박스 열 다음에 PID 값을 표시하는 새로운 셀(`<td>`)을 추가했습니다.
    *   이 셀에는 백엔드로부터 받은 데이터의 `t.pid` 값이 표시됩니다.

## 기대 효과
-   **향상된 추적성**: 사용자는 번역된 문장의 원본 식별자(PID)를 직접 확인할 수 있어, 원본 텍스트와의 대조 및 관리가 용이해집니다.
-   **디버깅 편의성**: 개발 및 디버깅 과정에서 특정 번역 항목을 식별하는 것이 더 명확해집니다.
