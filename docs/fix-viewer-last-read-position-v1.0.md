# 수정: 뷰어 마지막으로 읽은 위치 기억 기능

## 버전

v1.0

## 변경 사항

- 뷰어 페이지에서 마지막으로 읽은 위치를 기억하지 못하던 버그를 수정했습니다.
- 뷰어 진입 시, 서버에 저장된 마지막 읽은 위치(`last_read_pid`)를 불러와 해당 페이지를 표시합니다.
- 페이지를 넘길 때마다, 현재 페이지의 첫 번째 항목을 새로운 마지막 위치로 서버에 자동으로 저장합니다. (1.5초 디바운스 적용)
- 이 기능을 위해 `viewer.js`의 페이지네이션 및 렌더링 로직을 상당 부분 리팩토링했습니다.

## 변경된 파일

- `server/static/viewer.js`
