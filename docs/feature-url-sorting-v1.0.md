# Feature: 번역 기록 URL 숫자 기반 정렬 (v1.0)

## 기능 설명

번역 기록 페이지(`history.html`)에서 폴더 내에 있는 작품(URL) 목록의 정렬 방식을 개선했습니다. 기존의 단순 가나다순 정렬에서, URL의 마지막 숫자(에피소드 번호)를 기준으로 오름차순 정렬하도록 변경되었습니다.

## 구현 내용

1.  **`sortUrlsNumerically` 함수 추가**:
    *   URL 문자열에서 에피소드 번호를 추출하는 정규식을 사용하여 숫자 값을 가져옵니다.
    *   같은 작품(동일한 기본 URL)은 에피소드 번호 순서대로 정렬하고, 서로 다른 작품은 기본 URL의 가나다순으로 정렬하는 비교 로직을 구현했습니다.

2.  **`renderGroupedHistory` 함수 수정**:
    *   기존의 `Object.keys(urlGroups).sort()` 로직을 새로 구현한 `sortUrlsNumerically(Object.keys(urlGroups))`로 대체하여, 화면에 렌더링하기 전에 숫자 기반 정렬이 적용되도록 수정했습니다.

## 기대 효과

-   사용자가 번역 기록 페이지에서 작품의 회차를 1화부터 순서대로 쉽게 찾아볼 수 있어 사용 편의성이 크게 향상됩니다.
-   URL에 숫자가 없는 경우에도 기존의 가나다순 정렬이 유지되어 일관된 사용자 경험을 제공합니다.
