# 기능 추가: 특정 PID 패턴 번역에서 자동 제외 (v1.0)

## 변경 사항 요약
문단 식별자(PID)가 특정 패턴(la/lp + 숫자)과 일치하는 경우, 해당 문단을 번역 대상에서 자동으로 제외하는 메커니즘을 도입했습니다.

## 상세 내용

### `extension/content.js`
-   **`translateAllPTags` 함수 로직 수정**:
    -   번역할 문단을 수집하는 `paragraphs.forEach` 루프 내에 새로운 검사 로직을 추가했습니다.
    -   각 문단(`p`)의 `id`가 정규식 `/^(la|lp)\d+$/i` (즉, 'la' 또는 'lp'로 시작하고 하나 이상의 숫자로 끝나는 형식, 대소문자 무관)와 일치하는지 확인합니다.
    -   패턴이 일치하는 경우, 해당 문단을 번역 대상 목록(`textsToTranslate`)에 추가하지 않고 건너뛰도록 `return` 처리합니다.

## 기대 효과
-   **불필요한 번역 방지**: 광고나 기타 번역이 필요 없는 특정 영역을 번역 과정에서 자동으로 제외하여, 불필요한 API 호출을 줄이고 사용자가 더 깔끔한 번역 결과물을 얻을 수 있도록 합니다.
