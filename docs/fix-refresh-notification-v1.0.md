# 데이터 새로고침 알림 기능 복원 v1.0

## 변경 사항

- `history.js`의 `loadAndRender` 함수 실행 시, 데이터 로딩 상태를 알려주는 토스트 알림 기능이 누락된 문제를 해결했습니다.
- `loadAndRender` 함수의 시작, `fetchTitles` 호출 전, 그리고 함수 종료 시점에 각각 `showToast` 알림을 추가하여 사용자에게 데이터 처리 상태를 명확히 보여주도록 수정했습니다.

## 영향

- 이제 번역 기록 페이지 로드 시 사용자에게 데이터 처리 과정을 시각적으로 알려주어 사용성이 개선됩니다.
