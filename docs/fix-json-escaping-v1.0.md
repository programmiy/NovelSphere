## 버그 수정: API 응답 JSON 파싱 오류 해결 (v1.0)

### 변경 사항 요약

Gemini API가 반환하는 번역 결과물에 큰따옴표(`"`)가 포함될 경우, 이를 이스케이프 처리하지 않아 발생하는 JSON 파싱 오류를 해결했습니다.

API에 전달하는 프롬프트에 **"번역 결과에 큰따옴표가 포함되면 반드시 백슬래시로 이스케이프(`\"`)하라"**는 명확한 지침과 예시를 추가했습니다. 이를 통해 API가 항상 유효한 JSON 형식의 응답을 반환하도록 하여, 서버 측에서 안정적으로 데이터를 처리할 수 있게 만들었습니다.

### 주요 기술적 변경

-   `server.py`의 `translate_texts` 함수 내 프롬프트 문자열에 JSON 큰따옴표 이스케이프 관련 규칙을 명시적으로 추가했습니다.
