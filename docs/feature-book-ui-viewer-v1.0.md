# 기능: 뷰어 페이지에 '책 형태 UI' 적용

## 버전

v1.0

## 변경 사항

- 뷰어 페이지의 UI를 기존의 상하 스크롤 방식에서 실제 책처럼 페이지를 넘기는 방식으로 변경했습니다.
- 콘텐츠를 페이지 단위로 자동 분할하고, 이전/다음 버튼 및 키보드 화살표로 페이지를 넘길 수 있는 기능을 구현했습니다.
- 이 변경으로 인해 독서 경험이 향상될 것으로 기대됩니다.

## 변경된 파일

- `server/templates/viewer.html`
- `server/static/viewer.css`
- `server/static/viewer.js`
