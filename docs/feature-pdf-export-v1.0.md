### 주요 수정사항

*   번역 기록 페이지에 'PDF로 내보내기' 기능을 추가했습니다.
*   기존 '내보내기' 버튼은 'JSON으로 내보내기'로 명확하게 변경했습니다.
*   사용자가 UI에서 원하는 번역 기록(폴더별/개별)을 체크박스로 선택할 수 있습니다.
*   'PDF로 내보내기'를 누르면, 선택된 항목들이 책처럼 보기 좋은 2단 레이아웃의 새로운 HTML 페이지로 열립니다.
*   새 페이지의 '인쇄' 버튼을 눌러 브라우저의 'PDF로 저장' 기능을 통해 고품질의 PDF 파일을 생성할 수 있습니다.

---

### 보존할 세부 문서

**구현 전략:**
브라우저 환경에서 PDF를 직접 생성하려면 외부 라이브러리가 필요하지만, 현재 개발 환경의 제약으로 이를 추가할 수 없습니다. 이 문제를 해결하기 위해, PDF 생성 로직을 직접 구현하는 대신 브라우저의 내장 '인쇄' 기능을 활용하는 우회적인 방법을 채택했습니다.

`exportToPrintableHtml` 함수는 선택된 번역 데이터를 기반으로 인쇄에 최적화된 CSS 스타일이 적용된 HTML 문서를 동적으로 생성합니다. 이 HTML을 새 탭에서 연 후, 사용자는 `window.print()` 기능을 통해 직접 PDF로 저장하게 됩니다. 이 방식은 외부 의존성 없이 사용자의 요구사항을 효과적으로 만족시키는 실용적인 해결책입니다.
