# UI 개편 후 누락 기능 재구현 v1.0

## 변경 사항

UI 개편 과정에서 누락되었던 여러 주요 기능을 `history.js`와 `history.html`에 다시 구현했습니다.

- **URL 그룹 제목 수정 기능:** 각 URL 그룹 제목 옆의 수정 버튼(✏️)을 통해 제목을 직접 편집하고 저장할 수 있습니다.
- **페이지에 번역 적용/해제 토글:** 그룹 헤더의 '페이지에 적용' 버튼을 통해 특정 URL의 번역 내용을 페이지 로드 시 자동으로 적용할지 여부를 설정할 수 있습니다. 적용 시 버튼은 붉은색 '적용 해제' 상태로 변경됩니다.
- **URL별 찾아 바꾸기:** 그룹 헤더의 '찾아 바꾸기' 버튼을 통해 해당 URL의 번역 기록 내에서만 텍스트를 찾아 바꿀 수 있습니다.
- **개별 번역 제외 기능:** 각 번역 항목의 제외 버튼(🚫)을 통해 특정 문장을 이후 번역 작업에서 영구적으로 제외하고, 기록에서도 삭제합니다.
- **개별 번역 삭제 기능:** 각 번역 항목의 삭제 버튼(🗑️)이 정상적으로 동작하도록 누락되었던 `deleteTranslation` 함수를 복원했습니다.

## 영향

- 사용자는 이제 UI 개편 이전에 존재했던 풍부한 편집 및 관리 기능을 다시 사용할 수 있습니다.
- 번역 기록 관리의 편의성과 효율성이 크게 향상되었습니다.
