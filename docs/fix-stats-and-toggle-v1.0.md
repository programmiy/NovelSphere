# 통계 오류 및 기본 토글 상태 수정 v1.0

## 변경 사항

- **통계 오류 수정:**
  - 통계 수치가 모두 0으로 표시되던 문제를 해결하기 위해, 리팩토링 과정에서 누락되었던 `updateStats` 함수를 `history.js`에 다시 추가했습니다.

- **기본 토글 상태 확인:**
  - 페이지 로드 시 모든 그룹이 펼쳐져 있던 문제는 이전 작업에서 이미 수정되었음을 확인했습니다. `renderGroupedView` 함수는 이제 모든 그룹을 기본적으로 접힌 상태로 올바르게 렌더링합니다.

## 영향

- 총 번역, 오늘 번역, 평균 길이 통계가 정상적으로 계산되어 표시됩니다.
- 페이지 로드 시 모든 기록 그룹이 깔끔하게 접힌 상태로 표시됩니다.
