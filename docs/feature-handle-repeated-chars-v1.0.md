# 기능: 반복 문자 처리 기능 추가 (v1.0)

## 변경 사항

- 번역 API 응답 처리 과정에서 특정 문자가 4번 이상 연속으로 반복될 경우, 이를 `(반복)`으로 치환하는 기능을 추가했습니다.
- 이를 통해 비정상적인 번역 결과(예: `~~~~~~~~`)가 사용자에게 그대로 노출되는 것을 방지하고, 데이터의 가독성을 높입니다.

### 로직 추가 (server.py)

- `process_repeated_chars` 함수를 새로 추가하여 정규표현식(`(.)\1{3,}`)을 통해 반복 문자를 감지하고 치환합니다.
- `translate_texts` 함수 내에서 번역된 텍스트에 위 함수를 적용하여 최종 결과를 반환하도록 수정했습니다.
