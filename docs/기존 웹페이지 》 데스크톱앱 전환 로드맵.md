# Project LANOVEL - 데스크톱 앱 전환 로드맵

이 문서는 LANOVEL 웹 애플리케이션을 완전한 데스크톱 애플리케이션으로 전환하기 위한 개발 계획과 진행 상황을 추적합니다.

## 목표
- OS에 완전히 통합된 조용한 백그라운드 데스크톱 앱 경험 제공
- 터미널 의존성 제거 및 사용 편의성 극대화
- 앱의 상태와 활동에 대한 명확한 시각적 피드백 제공
