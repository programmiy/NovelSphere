## 폴더 생성 기능 수정 (v1.0)

### 요약

폴더 생성 기능이 작동하지 않던 문제를 해결했습니다. 서버에 폴더 생성을 위한 API 엔드포인트를 추가하고, 클라이언트에서 해당 엔드포인트를 호출하도록 로직을 수정했습니다.

### 변경 사항

- **`server/server.py`**
  - `POST /api/folders` 엔드포인트를 추가하여 새 폴더를 생성하는 기능을 구현했습니다.
  - `book_activity` 테이블에 폴더 이름을 추가하는 방식으로, 폴더가 존재하도록 처리합니다.

- **`extension/history.js`**
  - '폴더 생성' 버튼에 대한 클릭 이벤트 리스너를 추가했습니다.
  - `createFolder` 함수를 구현하여, 사용자가 입력한 폴더 이름으로 서버에 생성 요청을 보냅니다.
  - 폴더 생성 후, 페이지를 새로고침하여 변경 사항을 반영합니다.

- **`.gitignore`**
  - `.venv/` 디렉토리를 Git 추적에서 제외하도록 추가했습니다.
