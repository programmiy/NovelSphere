# 폴더 기능 복원 및 UI 가독성 개선 v1.0

## 변경 사항

- **폴더 기능 복원:** UI 개편으로 누락되었던 폴더별 그룹화 기능을 복원했습니다. 이제 번역 기록이 폴더를 기준으로 그룹화되어 표시됩니다.
  - `loadAndRender` 함수의 데이터 처리 로직을 폴더 기준으로 변경했습니다.
  - `renderView` 함수의 이름을 `renderGroupedView`로 변경하고, 폴더 그룹을 렌더링하도록 전체 로직을 수정했습니다.
  - 폴더 그룹 UI를 위한 CSS 스타일과 토글 기능을 위한 이벤트 리스너를 추가했습니다.

- **UI 가독성 개선:** 그룹 헤더의 '페이지에 적용', '찾아 바꾸기' 버튼이 눈에 잘 띄지 않는 문제를 해결했습니다.
  - `.btn-secondary`라는 새로운 버튼 스타일을 정의하여 기존의 `.btn-outline`보다 시각적으로 강조되도록 했습니다.
  - 해당 버튼들에 새 스타일을 적용하여 가독성을 높였습니다.

## 영향

- 사용자는 폴더를 이용해 번역 기록을 체계적으로 관리하고 조회할 수 있습니다.
- 주요 기능 버튼들이 더 명확하게 보여 사용자 경험(UX)이 개선되었습니다.
