# 기능: 서버리스 HTML 책 내보내기 (v1.0)

## 개요

특정 책(폴더)의 모든 번역 데이터와 뷰어 UI를 하나의 독립된 HTML 파일로 생성하여 내보내는 기능을 추가했습니다. 이 파일은 서버 구동 없이 로컬에서 바로 열어 책의 내용을 확인할 수 있습니다.

## 주요 로직

1.  **API 엔드포인트 추가**:
    *   `GET /api/books/{folder_name}/export` 엔드포인트를 추가했습니다.

2.  **데이터 및 리소스 통합**:
    *   요청된 책의 모든 번역 데이터를 데이터베이스에서 조회합니다.
    *   뷰어 UI에 필요한 `viewer.css`와 `viewer.js` 파일의 내용을 읽어옵니다.

3.  **독립 HTML 파일 생성**:
    *   `export_template.html` 템플릿을 사용하여, 조회된 책 데이터(JSON), CSS, JS 내용을 한 파일에 모두 삽입(inline)합니다.
    *   템플릿 내부의 스크립트는 API를 호출하는 대신, 파일 내에 주입된 JSON 변수에서 직접 데이터를 읽어 뷰어를 렌더링합니다.

4.  **다운로드 제공**:
    *   생성된 HTML 콘텐츠를 `Content-Disposition: attachment` 헤더와 함께 `HTMLResponse`로 반환하여, 사용자가 파일로 다운로드할 수 있도록 합니다.
